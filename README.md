# EXIF 복사기

## 설치 환경 및 방법

- 설치 환경

  - Windows and Linux

  - Python 3

- 설치 방법

  - [Python 3](https://www.python.org/downloads/)

    - (Windows) 설치 중간에 나오는 `Add to PATH` 체크 필수!

  - [ExifTool by Phil Harvey](https://exiftool.org/)

    - (Windows) Windows Executable: exiftool-xx.xx.zip 을 다운로드!
